# Privacy Policy

Last updated: 2025-01-01

This extension is intended for use with University of Tennessee's ezyVet at `https://utcvm.use1.ezyvet.com/*`.

## Data the extension accesses
- Page content from the allowed ezyVet domain to extract lab results and related metadata.

## Data the extension stores
- Extracted lab trends and related metadata (e.g., patient name/ID, owner last name, panel/test results).
- UI preferences (toggles and options).

Stored data is kept locally in the browser via Chrome storage (`chrome.storage.local`, `chrome.storage.session`, and `chrome.storage.sync`).

## Data sharing
- The extension does not transmit data to external servers.
- No analytics or tracking services are used.

## Data deletion
- You can delete stored lab trends using the “Clear” button in the side panel.
- You can also remove the extension to delete its local data.
